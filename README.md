# Iran Alert Bot

Bot de Telegram 100% gratis que te avisa SOLO cuando hay eventos clave en la guerra Irán-EE.UU.-Israel.

### Cómo funciona
- Revisa cada 8 minutos fuentes confiables (Reuters, AP, BBC, Al Jazeera, NYT).
- Detecta automáticamente ataques, misiles, bajas o escaladas importantes.
- Te manda una sola alerta clara cuando pasa algo real (sin spam).

### Tecnologías
- Python + aiogram
- Deploy en Railway
